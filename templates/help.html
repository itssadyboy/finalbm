User Roles:
  • Admin: Full access to all features
  • User: Can only create production entries and manage masters
Masters Management:

Add all necessary master data before creating entries:

  • Operators: Production staff
  • Parties: Customers for sales
  • Machines: Production equipment
  • Items: Products (mark as Production or Sale type)
